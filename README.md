# 🌱 Carbon-Aware ML Training Controller

A real-time carbon-aware scheduling system that optimizes machine learning training workloads based on UK electricity grid carbon intensity.

It helps reduce CO₂ emissions by scheduling compute tasks during low-carbon periods.

---

## 🚀 Key Features

- 📡 Real-time UK grid carbon intensity integration
- 🧠 Carbon-aware scheduling engine (optimal execution window detection)
- 🧪 ML workload simulation (energy + emissions estimation)
- 📊 Interactive Streamlit dashboard
- 📈 Forecast exploration & peak detection
- ⚙️ Simulation lab for workload-carbon analysis

---

## 🧱 System Architecture

1. **Carbon Intelligence Layer**
   - Fetches UK National Grid carbon intensity data

2. **Scheduling Engine**
   - Finds optimal training windows
   - Minimizes carbon exposure

3. **ML Workload Simulator**
   - Simulates training energy usage
   - Estimates emissions impact

4. **Analytics Dashboard**
   - Multi-page Streamlit interface
   - Interactive visualizations (Plotly)
